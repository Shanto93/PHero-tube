Discuss the scope of var, let, and const
  • var: Variables declared with var have function scope. This is the default scope for variables in JavaScript.
  • let: Variables declared with let have block scope.
  • const: Variables declared with const have block scope and cannot be reassigned.
Tell us the use cases of null and undefined
  • Null is used to represent the intentional absence of a value. For example, you might use null to represent a missing value in a database or an empty object.
  • Undefined is used to represent the unintentional absence of a value. For example, you might get an undefined value if you try to access a variable that has not been assigned a value.
What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing and developing web services. REST APIs are based on the HTTP protocol and use standard HTTP methods to access resources.
